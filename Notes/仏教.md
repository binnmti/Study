# 仏教史の流れ
- 初期仏教: 釈迦が説いた教え
  - 仏陀の言葉と律
- 部派仏教: 釈迦の教えを守る過程で分派し、教義や戒律が多様化。
  - 上座部は、戒律や教義を厳密に守る立場。
  - 大衆部は、戒律を柔軟に解釈し、大衆的な活動を重視。
  - アビダルマ（論蔵）を体系化したのも、この部派仏教の時代
- 大乗仏教: 部派仏教(特に大衆部)を基盤に発展し、より多くの人を救うために仏や菩薩を理想化。
  - ただし、単純に大衆部が部派仏教になったわけではない。上座部の影響もうけている。もっと複雑。
  - 大乗仏教は特定の宗派の単独の流れから生まれたのではなく、複数の思想や運動が交わる中で形成された。

## 部派仏教の時代と多様性
- 仏教がインドで広まり、さまざまな解釈や実践が生まれた
- 上座部仏教（Theravāda）や説一切有部（Sarvāstivāda）、法蔵部（Mahīśāsaka）など
- 仏教の基本的な教義はまだ比較的一貫しており、仏陀の教えをそのまま守ることが重要視されてた
- しかし、教義や修行のアプローチに違いが現れ、教義が多様化していった

## 大乗仏教の誕生
- 「普遍的な仏教の教え」を強調する新たな宗派が台頭してきた
- この時期に登場した思想や経典は、「大乗経典」と呼ばれ、仏陀の教えを広く一般の人々に解放的に開放することを目指した
- 「大乗」（Mahāyāna）は、「大きな車輪」や「広い道」を意味し、解脱に至る道を広く開放する、あるいは広い人々に向けて教えを説くこと
- 「小乗」（Hīnayāna）は、それに対して、修行僧や特定の信者を対象にした教えを指す言葉
- 「大乗」や「小乗」という言葉は大乗仏教側からの呼称。自己をより広範な人々に開かれた仏教の教えとして位置づけるために使われた

## 大乗仏教経典の誕生
- 大乗仏教が形成される過程で、新しい経典や教義が登場した
- これらの経典には、「菩薩の道」や「仏の神格化」、さらには「仏陀の教えがすべての存在に共通している」という考え方が盛り込まれた
- 有名な経典『法華経』『華厳経』『大般若経』
- これらの経典は、従来の部派仏教（小乗仏教）とは異なる、より普遍的で大衆的な救済の道を描いている。
- 中国や日本、チベットなどで仏教が広まる過程で、「大乗仏教」という名称が広まり、次第に仏教の一つの大きな流派として認識されるようになった

## 「大乗仏教」の広がり
- 大乗仏教は、インドを中心に広まり、後に中国、韓国、日本、チベットなどに伝播していった
- 特に中国では、禅宗や浄土宗、法華宗など、さまざまな宗派が大乗仏教の思想を発展させていった
- また、日本では、浄土宗や真言宗など、仏陀や菩薩の神格化が進んだ

# 神格化の進展
- 大乗仏教が発展する過程で、仏教はインドの宗教的文化と密接に関わり、他の宗教や信仰体系からの影響を受けた。
- 特にヒンドゥー教やバラモン教、さらにはゾロアスター教などの他の宗教で見られる神格化の概念が影響を与えた。
- 大乗仏教の発展とともに、仏陀や菩薩が「神のような存在」として認識されるようになった
- 仏教教義の中で「仏の普遍的な性質」が強調されるようになったため
　　- 仏教が持つ悟りの概念や空の教義が重要な役割を果たしている
  
## 仏陀の普遍性
- 大乗仏教では、仏陀が単なる歴史的な人物であるという枠を超えて、悟りそのものを象徴する存在として理解されるようになった
- 仏陀は、すべての存在に共通する普遍的な真理を体現する存在として認識され、次第に神格化していく
- これにより、仏陀はただの教師から、信仰の対象であり、他者を救う存在（救済者）として崇拝されるようになりました。

## 菩薩の登場と神格化
- 大乗仏教では、菩薩という新しい概念が登場する。
- 菩薩は、自己の悟りよりも他者の救済を優先する存在として描かれ、特に観音菩薩や地蔵菩薩など、信仰の対象として神格化されることが多くなった。
- これらの菩薩は、人々を助け、守護する役割を果たし、神的存在としての位置を確立していきました。

## 仏の不死性や神的役割
- 大乗仏教の中で、仏陀や菩薩は次第に永遠不滅で不死であると考えられるようになり、その結果、神格化の色彩が強くなった。
- 特に、浄土宗や浄土真宗のような流派では、阿弥陀仏が信者を極楽浄土に導く唯一の存在として、神的な役割を担うことになった。  

# アビダルマ
- 仏教の教えを分類し、分析し、体系化するための文献群
- 仏陀の教えを受け継いだ弟子たちは、その教えを様々な地域や文化の中で広め、部派仏教という異なる解釈や実践が生まれた。
- この時期に、仏教徒は教義をさらに発展させ、哲学的な体系や論理的な分析が必要だと感じるようになった
- アビダルマは、教義をさらに体系的に分析し、詳細に解釈するために誕生した。
- 心理学、倫理学、形而上学、そして論理学などの領域が含まれる
- 仏教は単なる修行の道から、より複雑で知的な学問としての側面も持つようになった

# 実践から理論へのシフト
- 仏陀は、理論よりも実際の行動変化や体験を重視
- 学問的な仏教が発展し、瞑想だけでなく知識の習得も重要視されるようになった
- 教義の抽象化：仏陀の教えを全ての人に適用できる普遍的な理論として再構築。
- これが経典の異なる解釈間での対立を緩和する役割を果たした
- 瞑想の目的と方法が多様化し、哲学的な瞑想も実践されるようになった
- アビダルマは部派仏教の各派で独自に発展し、派ごとに内容や解釈が異なる

# 神格化していった理由
- アビダルマで「悟り」という理想像が非常に高度で抽象的なものとして示された
- 仏陀自身も人間として到達可能な目標以上に、超越的な存在としての仏陀のイメージを強化した
- 学問的整理が仏陀を「普通の人間以上の存在」として理解させる要因になった
- アビダルマは高度に哲学的・学問的なため、民衆は理解が難しかった
- 民衆に広がる過程で、仏陀を超越的な存在として描く方が効果的だった
- 具体的な形で表現できる崇拝の対象として仏陀の神格化を促す
- また仏教がインドで発展する過程で、既存の宗教であるヒンドゥー教やバラモン教の影響を受ける
- これらの宗教には既に超越的な神々への崇拝があり、それと類似の性質を仏陀に与えることで、仏教の信仰が親しみやすくなる
- 菩薩や仏陀が超越的な力を持つ存在として描かれ、特に「阿弥陀仏」や「観音菩薩」などの崇拝対象が神格化された

# 神話の始まりは？
- シュメール神話（メソポタミア）
  - シュメール文明（紀元前3500年～紀元前2000年頃）は、最も古い記録が残っている文明のひとつ。
  - シュメール人は楔形文字を用いて神話や歴史を記録した。
  - シュメール神話の中でも最も有名なものが、ギルガメシュ叙事詩
 
- エジプト神話
  - 古代エジプト文明も紀元前3000年頃から存在している。
  - 神々や宇宙の創造、死後の世界に関する深い理解が反映されている。
  - エジプトの神話では、ラー（太陽神）やオシリス（死と復活の神）が重要な役割。

- インダス文明
  - インダス文明（紀元前3300年～紀元前1300年頃）は、現在のインドとパキスタンにまたがる地域に存在した古代文明。
  - その宗教的な概念は後のインドのヴェーダ神話やヒンドゥー教に影響を与えた。
  -  動物や自然現象に対する崇拝が行われ、神々や精霊の信仰が存在したと推測される
  -  後のヒンドゥー教で重要な役割を果たす神々（シヴァなど）への信仰が示唆されている

- インドのヴェーダ神話
  - インダス文明に続くインドのヴェーダ時代（紀元前1500年～紀元前500年）
  - ヒンドゥー教の基盤となる神話や宗教的な文献が登場した時期。
  - ヴェーダ神話は、後のヒンドゥー教の神話に多大な影響を与えています。

- 中国の古代神話
  - 中国の古代神話は、紀元前2000年頃の古代中国文明に遡る。
  - 最も古い神話や伝説は、紀元前12世紀頃の殷朝（紀元前1600年～紀元前1046年）やその前の夏朝（紀元前2070年～紀元前1600年）に関するもの。
  - 盤古神話: 世界創造に関する神話で、盤古が天地を分けて創造したという物語がある。

# 神と言う概念の進化
自然神信仰（アニミズム）→ 多神教 → 一神教 → 哲学的・抽象的な神 → 多様化した神の概念

## 哲学的・抽象的な神
- スピノザの汎神論（すべてが神である）
- デカルト・カントのような哲学者たちは、神を「物理法則の根源」と捉える。
- 近代では「神は存在しない」とする無神論
- 「神の存在は証明できない」とする不可知論

## 現代の神の概念
- 科学者の中には、「宇宙の法則そのものが神だ」と考える人もいる。
- 心理学的には、「神は人間の意識が生み出したもの」と解釈されることもある。
- AIやシンギュラリティ（技術的特異点）の文脈で、「未来の超知能が神のような存在になるかもしれない」という考えもある。

# 多神教から一神教になった流れ
## 王の権力強化 → 神の統一
- 古代社会では、多神教が一般的。王の権力が強まると「王が唯一の正統な神の代理人」として振る舞うようになったことが、一神教の発展に影響。
## 民族宗教の発展
- 特定の民族の神が普遍化していくことで、一神教の発展が促された。

## 社会の統合と帝国の形成
- 多神教ではなく、統一宗教の方が統治しやすい
- 多神教の社会では、各地に異なる神々が存在し、それぞれに独自の祭祀や儀式があった。
- しかし、大規模な帝国（例：ペルシア帝国、ローマ帝国など）が成立すると、多様な文化や民族を統治するために「統一された宗教」が必要になった。
- 一神教の方が、**「唯一の神に従うことで、唯一の王（皇帝）に従う意識を強める」**という政治的な利点を持っていた。

## 哲学的・倫理的な変化
- 多神教の神々は、人間に似た性格（嫉妬、怒り、恋愛など）を持っていましたが、次第に「神はもっと絶対的で超越的な存在であるべき」という考えが生まれた。
- この思想は、後のユダヤ教、キリスト教、イスラム教に影響を与えた。

# 一神教の発展は、支配者が統治しやすい仕組みとして機能した側面がある
- 絶対的な権威を作れる
- 民衆を従わせやすい
- 支配者が「神の代理人」になれる
- 反乱や異端を抑えやすい
- 帝国の統治に向いていた

# 神は人が作ったものか？
- 神の概念は人間の必要に応じて変化してきた
- ★「神は人が作った」と考えることはできても、神という概念が無意味だったとは言えない。

# 宗教以外の神の側面
- 哲学的な「神」
  - デカルトの「完全なる存在としての神」
  - スピノザの「神即自然（パンセティズム）」
  -  ニーチェの「神は死んだ」
- 科学的な「神」
  - アインシュタインの「神」
  - ホーキングの「宇宙創造の神」
-  芸術・創造の「神」
  - 天才や偉業を「神」と称する
  - インスピレーションの源としての神
- 社会・文化の「神」
  - 国家や権力の正当化としての神
  - 歴史上の偉人（ナポレオン、リンカーンなど）が「神のように扱われる」
- 心理学・スピリチュアルな「神」
  - ユングの「元型としての神」
  - マズローの「至高体験（ピーク・エクスペリエンス）」
- 神 = 宇宙の法則（科学）
- 神 = 天才的な才能（芸術・スポーツ）
- 神 = 権力の正当化（政治・歴史）
- 神 = 精神的な象徴（心理学・スピリチュアル）
