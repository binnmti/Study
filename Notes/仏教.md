# 阿弥陀如来とは？
- 釈迦如来（ゴータマ・シッダールタ）は、仏教の創始者であり、悟りを開いたブッダとして知られています
- 釈迦如来と阿弥陀如来は別の仏ですが、両方ともブッダです。
- ゴータマ・シッダールタは阿弥陀如来ではない。
- この辺ややこしい。。

# [阿弥陀如来と釈迦如来とは](https://www.youtube.com/watch?v=w22Sr3Z9eWg)
- 阿弥陀ブッダとも言っていた
- 両方とも如来である
- 如来は = 仏様。
- 阿弥陀仏とも釈迦仏とも言う
- 仏 = 悟りのこと
- 悟りは52段階あって最高の悟りを仏覚と言う
- 52段階の内51段階までは全て菩薩と言う
- 仏の悟りを開いたことをブッダとも言う
- つまり仏の悟り = ブッダ = 如来ということ。
- 釈迦以外で悟りを開くのは弥勒菩薩で56億7000万年後にしか登場しない
- 弥陀如来は三世十法の諸仏の本師本仏なり
- 弥陀如来は阿弥陀如来のこと
- 阿弥陀仏の力を他力という
- 阿弥陀如来が師匠で、お釈迦様が弟子。阿弥陀如来が先生で、お釈迦様が生徒。

# 阿弥陀如来と釈迦如来の違いが分からない。
- ゴータマ・シッダールタは、釈迦如来としても知られる実際の歴史的人物
- 彼が悟りを開いて仏（ブッダ）となった。
- この「ブッダ」は、すべての人々が悟りを得る可能性を持つという意味で、一般的には「釈迦如来」と同義とされています。
- 阿弥陀如来（アミダ仏）は、大乗仏教における仏
- 釈迦如来（ゴータマ・シッダールタ）は、実際の歴史的な仏教の創始者であり、仏教の教えを直接的に説いた。
- この二人の仏が同じ「ブッダ」とされる点は、両者が仏教の教えにおいて「完全に覚醒した存在」であるという共通点から来ている
- ↑結論として、矛盾しているわけではなく、宗派や教義により強調される側面が異なるだけ
